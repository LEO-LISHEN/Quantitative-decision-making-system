# 检查清单

## 基础信息

- [ ] 公司名称、股票代码、交易市场、交易所已确认。
- [ ] 分析日期和信息截止时间已确认。
- [ ] 分析周期已确认。
- [ ] 数据模式已确认：上传表格 / API / 截图 / 文字描述。

## 数据质量

- [ ] 已获取日 K 数据。
- [ ] 日 K 数据至少包含 open/high/low/close/volume。
- [ ] 已获取成交额或说明缺失。
- [ ] 已确认复权方式。
- [ ] 已检查停复牌。
- [ ] 已检查除权除息、拆股、配股或合股。
- [ ] 已检查异常成交量和异常价格。
- [ ] 已确认数据长度是否足够。
- [ ] 已输出数据质量评级。

## 基准和事件

- [ ] 已获取大盘指数。
- [ ] 已获取行业指数、主题 ETF 或可比公司数据。
- [ ] 已获取事件时间线。
- [ ] 已确认事件日期和来源。
- [ ] 已说明没有基准或事件数据时的限制。

## 趋势判断

- [ ] 已判断主要趋势状态。
- [ ] 已判断短期趋势。
- [ ] 已判断中期趋势。
- [ ] 已检查高点和低点变化。
- [ ] 已检查价格所处历史区间。

## 均线结构

- [ ] 已检查短期均线。
- [ ] 已检查中期均线。
- [ ] 已检查长期均线，如数据足够。
- [ ] 已判断多头、空头、粘合或混乱。
- [ ] 已判断价格相对关键均线位置。

## 量价关系

- [ ] 已判断上涨时是否放量。
- [ ] 已判断下跌时是否放量。
- [ ] 已检查成交额质量。
- [ ] 已检查换手率或成交活跃度。
- [ ] 已识别异常成交日期。
- [ ] 已判断是否存在量价背离。

## 支撑阻力

- [ ] 已识别前高前低。
- [ ] 已识别缺口。
- [ ] 已识别成交密集区或平台。
- [ ] 已识别关键支撑区域。
- [ ] 已识别关键阻力区域。
- [ ] 已给出技术失效位或失效区域。

## 突破和跌破

- [ ] 已判断是否突破。
- [ ] 已判断是否跌破。
- [ ] 已判断成交量是否确认。
- [ ] 已判断是否有后续延续。
- [ ] 已提示假突破或假跌破风险。

## 相对强弱

- [ ] 已比较个股和大盘。
- [ ] 已比较个股和行业。
- [ ] 如有数据，已比较个股和可比公司。
- [ ] 已判断相对强弱改善、恶化或稳定。

## 事件后反应

- [ ] 已对齐事件日期和交易日期。
- [ ] 已分析事件后价格反应。
- [ ] 已分析事件后成交量反应。
- [ ] 已判断利好不涨、利空不跌、跳空、冲高回落或放量破位。

## 输出边界

- [ ] 已输出技术确认状态。
- [ ] 已输出择时含义。
- [ ] 已输出技术失效信号。
- [ ] 已列出需要补充的数据。
- [ ] 已列出需要其他分析师复核的问题。
- [ ] 没有直接给最终买入、卖出或持有建议。
