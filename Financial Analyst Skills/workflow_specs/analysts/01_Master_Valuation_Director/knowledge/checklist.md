# 检查清单

## 基础任务

- [ ] 公司名称、股票代码、市场、交易所已确认。
- [ ] 分析日期和信息截止时间已确认。
- [ ] 用户目标已确认。
- [ ] 用户持仓状态已确认或标注未知。
- [ ] 用户关注周期已确认。
- [ ] 当前价格、市值和货币已确认。

## 信息质量

- [ ] 已读取信息源分析师输出。
- [ ] 已确认来源等级。
- [ ] 已确认缺失信息。
- [ ] 已确认未验证传闻。
- [ ] 已确认信息截止日期。
- [ ] 已根据来源质量调整置信度。

## 专项分析师整合

- [ ] 已整合基本面分析。
- [ ] 已整合财务质量分析。
- [ ] 已整合行业周期分析。
- [ ] 已整合成长股分析。
- [ ] 已整合盈利预测修正。
- [ ] 已整合 DCF 估值。
- [ ] 已整合相对估值。
- [ ] 已整合市场预期差。
- [ ] 已整合事件催化剂。
- [ ] 已整合技术确认。
- [ ] 已整合舆情分析。
- [ ] 已整合风险反证。

## 估值

- [ ] 已判断 DCF 是否适用。
- [ ] 已判断相对估值是否可靠。
- [ ] 已考虑 SOTP、概率加权、股息模型或资产价值是否适用。
- [ ] 已给出悲观、基准、乐观情景。
- [ ] 已给出最终估值区间。
- [ ] 已给出相对当前价格的潜在涨跌幅。
- [ ] 已说明关键敏感假设。

## 分周期建议

- [ ] 已给出短期建议。
- [ ] 已给出中期建议。
- [ ] 已给出长期建议。
- [ ] 每个周期都有预期兑现时间。
- [ ] 每个周期都有理由。
- [ ] 每个周期都有失效信号。
- [ ] 如果周期建议冲突，已解释原因。

## 风险和反证

- [ ] 已列出主要风险。
- [ ] 已列出证伪条件。
- [ ] 已判断是否触发风险否决。
- [ ] 已说明下行情景。
- [ ] 已列出最终建议前必须核验的事项。

## 最终建议

- [ ] 已给出总体建议。
- [ ] 已给出建议置信度。
- [ ] 已说明风险收益比。
- [ ] 已说明建议成立前提。
- [ ] 已说明跟踪指标。
- [ ] 已说明不保证收益或股价方向。
- [ ] 信息不足时未给出高置信买入建议。
