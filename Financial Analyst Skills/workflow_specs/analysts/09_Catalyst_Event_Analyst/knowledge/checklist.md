# 检查清单

## 数据完整性

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认信息截止时间？
- [ ] 是否有事件来源？
- [ ] 是否有预计时间？
- [ ] 是否有事件可信度？
- [ ] 是否有市场定价状态？
- [ ] 是否有影响测算或影响路径？

## 催化剂识别

- [ ] 是否区分普通新闻和真正催化剂？
- [ ] 是否判断事件能否影响现金流？
- [ ] 是否判断事件能否影响盈利预测？
- [ ] 是否判断事件能否影响风险溢价？
- [ ] 是否判断事件能否影响估值倍数？
- [ ] 是否判断事件能否影响股东回报？
- [ ] 是否判断事件能否影响资金流？

## 分类

- [ ] 是否列出正面催化剂？
- [ ] 是否列出负面催化剂？
- [ ] 是否列出不确定催化剂？
- [ ] 是否标记非催化剂？

## 四要素

- [ ] 是否评估发生概率？
- [ ] 是否评估时间窗口？
- [ ] 是否评估影响强度？
- [ ] 是否评估是否已定价？

## 强度评级

- [ ] 是否给出强、中、弱、非催化剂或负面催化剂评级？
- [ ] 已充分定价的事件是否避免标记为强催化剂？
- [ ] 无来源或无时间的事件是否标记为待验证？

## 确认和失败信号

- [ ] 是否列出确认信号？
- [ ] 是否列出失败信号？
- [ ] 是否列出后续跟踪指标？

## 跨市场

- [ ] A 股是否关注业绩预告、政策、订单、重组、限售解禁、龙虎榜、机构调研和国企改革？
- [ ] 港股是否关注分红、回购、私有化、配股供股、南向资金、沽空和盈利警告？
- [ ] 美股是否关注 earnings、guidance、FDA、investor day、buyback、M&A、index inclusion 和 SEC filing？
- [ ] 中概股是否关注监管、审计、ADR/港股双重上市、VIE 和地缘风险？

## 输出

- [ ] 是否输出催化剂日历？
- [ ] 是否输出价值兑现路径？
- [ ] 是否说明需要其他分析师验证的问题？
- [ ] 是否避免直接给最终买卖建议？
