# 检查清单

## 基础信息

- [ ] 公司名称、股票代码、市场和交易所已确认。
- [ ] 舆情分析日期已确认。
- [ ] 舆情时间窗口已确认。
- [ ] 语言范围已确认。
- [ ] 重点平台已确认。

## 来源覆盖

- [ ] 已检查股吧或本地财经社区。
- [ ] 已检查社交媒体。
- [ ] 已检查新闻和新闻评论。
- [ ] 已检查 KOL 或专业投资者观点。
- [ ] 已检查搜索热度或讨论热度，如可得。
- [ ] 已检查公司公告、回应或正式披露。
- [ ] 已检查做空报告或负面专题，如存在。
- [ ] 已标注样本量和时间范围。

## 来源甄别

- [ ] 已区分事实、观点、情绪和传闻。
- [ ] 已标注每类来源可信度。
- [ ] 已记录发布时间。
- [ ] 已记录传播范围或互动量，如可得。
- [ ] 已识别匿名爆料、截图或群聊消息。
- [ ] 已识别 KOL 利益倾向或持仓偏见。
- [ ] 已说明样本偏差。

## 叙事分析

- [ ] 已提取主流看多叙事。
- [ ] 已提取主流看空叙事。
- [ ] 已提取反对叙事。
- [ ] 已提取新兴话题。
- [ ] 已提取高频关键词。
- [ ] 已识别公告或财报的常见误读。
- [ ] 已判断叙事是否有正式证据支持。

## 情绪判断

- [ ] 已判断整体情绪状态。
- [ ] 已判断情绪变化方向。
- [ ] 已判断讨论热度。
- [ ] 已判断拥挤度。
- [ ] 已判断分歧度。
- [ ] 已说明置信度。

## 传闻和风险

- [ ] 已列出未证实信息。
- [ ] 已标注传闻来源和传播范围。
- [ ] 已说明需要核验的材料。
- [ ] 已识别声誉风险。
- [ ] 已识别监管传闻或合规风险。
- [ ] 已识别做空报告或负面爆料风险。

## 短期价格影响

- [ ] 已判断短期波动风险。
- [ ] 已判断可能方向或方向不明。
- [ ] 已判断可能时间窗口。
- [ ] 已识别放大渠道，例如 KOL、新闻标题、期权、涨跌停或低流动性。
- [ ] 已列出需要技术分析师对照的时间点。

## 联动输出

- [ ] 已列出交给信息源分析师核验的问题。
- [ ] 已列出交给预期差分析师的问题。
- [ ] 已列出交给技术分析师的问题。
- [ ] 已列出交给催化剂分析师的问题。
- [ ] 已列出交给风险分析师的问题。
- [ ] 没有直接给最终买入、卖出或持有建议。
