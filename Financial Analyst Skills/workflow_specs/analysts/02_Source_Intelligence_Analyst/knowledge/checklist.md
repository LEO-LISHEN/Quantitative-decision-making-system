# 检查清单

## 公司识别

- [ ] 是否确认公司名称？
- [ ] 是否确认股票代码？
- [ ] 是否确认上市市场？
- [ ] 是否确认交易所？
- [ ] 是否确认适用的信息源体系：A 股、港股、美股或其他？
- [ ] 是否确认币种？
- [ ] 是否确认会计准则？
- [ ] 是否确认分析时间范围？
- [ ] 是否标注信息截止时间和时区？

## 用户材料盘点

- [ ] 是否列出所有用户上传或粘贴的材料？
- [ ] 是否标注每份材料的来源？
- [ ] 是否标注发布日期或事件日期？
- [ ] 是否区分研报、新闻、公告、财报、论坛和图表？

## 主动搜集

- [ ] 是否判断当前平台是否具备搜索或数据工具？
- [ ] 如果可以搜索，是否优先查找公告、财报、监管披露？
- [ ] 是否使用了对应市场的信息源，例如巨潮、披露易或 SEC EDGAR？
- [ ] 是否补充近期重大新闻和行业事件？
- [ ] 是否补充市场情绪或股价反应信息？
- [ ] 如果不能搜索，是否明确说明限制？
- [ ] 是否给出建议搜索来源和关键词？
- [ ] 如果搜索失败，是否说明已尝试搜索什么、未找到什么和可能原因？

## 证据分级

- [ ] 是否使用 A/B/C/D/E 或等价方式区分证据等级？
- [ ] 是否把股吧和社交媒体内容限定为情绪证据？
- [ ] 是否标记了传闻？
- [ ] 是否标记了不同来源之间的矛盾？
- [ ] 是否避免把研报观点当成事实？
- [ ] 是否避免把预测当成已发生事实？

## 口径检查

- [ ] 是否检查币种？
- [ ] 是否检查单位？
- [ ] 是否检查报告期？
- [ ] 是否检查会计准则？
- [ ] 是否检查复权或不复权价格口径？
- [ ] 是否检查跨市场、多地上市或 ADR/H 股差异？

## 估值变量

- [ ] 是否提取收入相关信息？
- [ ] 是否提取利润率相关信息？
- [ ] 是否提取订单、产能、价格、成本信息？
- [ ] 是否提取现金流和资产负债信息？
- [ ] 是否提取政策、行业和周期信息？
- [ ] 是否提取资本运作、回购、分红、融资信息？
- [ ] 是否提取重大风险事件？

## 下游传递

- [ ] 是否建议调用后续分析师模块？
- [ ] 是否明确哪些信息适合传给哪个模块？
- [ ] 是否保留了未验证信息的标签？
- [ ] 是否避免输出目标价和投资评级？
