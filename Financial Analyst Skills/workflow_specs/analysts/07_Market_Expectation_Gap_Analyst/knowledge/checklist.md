# 检查清单

## 数据完整性

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认信息截止时间？
- [ ] 是否有事件发生时间？
- [ ] 是否有信息公开时间？
- [ ] 是否有新闻、公告或研报发布时间？
- [ ] 是否有事件前后日 K 线？
- [ ] 是否有成交量和成交额？
- [ ] 是否有大盘和行业同期表现？
- [ ] 是否有一致预期或市场叙事？

## K 线表格

- [ ] 是否包含 date？
- [ ] 是否包含 open、high、low、close？
- [ ] 是否包含 volume？
- [ ] 是否包含 turnover？
- [ ] 是否包含 pct_change？
- [ ] 是否包含 adjusted_close 或说明复权口径？
- [ ] 是否包含 market_index_pct_change？
- [ ] 是否包含 sector_index_pct_change？

## 信息新旧判断

- [ ] 是否区分全新信息？
- [ ] 是否区分旧信息确认？
- [ ] 是否区分已广泛预期？
- [ ] 是否区分低于预期？
- [ ] 是否区分噪音信息？
- [ ] 是否避免把用户刚知道的信息当成新信息？

## 信息重要性

- [ ] 是否判断信息是否影响收入？
- [ ] 是否判断信息是否影响利润率？
- [ ] 是否判断信息是否影响现金流？
- [ ] 是否判断信息是否影响风险溢价？
- [ ] 是否判断信息是否影响估值倍数？
- [ ] 是否判断信息是否影响催化剂概率？

## 定价状态

- [ ] 是否判断未定价、部分定价、基本定价或过度定价？
- [ ] 是否检查公告前异动？
- [ ] 是否检查公告后反应？
- [ ] 是否检查成交量变化？
- [ ] 是否检查相对大盘和行业表现？
- [ ] 缺少 K 线时是否降低置信度？

## 预期差

- [ ] 是否判断方向：正向、负向、中性、混合？
- [ ] 是否判断强度：强、中、弱、无、无法判断？
- [ ] 是否说明持续性：短期扰动、中期变化、长期变化？
- [ ] 是否列出未来继续定价路径？
- [ ] 是否列出反证信号？

## 跨市场

- [ ] A 股是否关注涨跌停、龙虎榜、题材、机构调研、北向资金和融资融券？
- [ ] 港股是否关注南向资金、成交额、沽空、回购分红和流动性不足？
- [ ] 美股是否关注盘前盘后、guidance、analyst revision、options 和 sector ETF？
- [ ] 多地上市公司是否分别观察不同市场反应？

## 输出

- [ ] 是否先说明数据是否足够？
- [ ] 是否明确缺失数据和补充要求？
- [ ] 是否避免直接给最终买卖建议？
- [ ] 是否给总控分析师使用建议？
