# 方法论

## 1. 先判断有没有周期

行业周期分析不能从“默认有周期”开始。  
第一步是判断行业是否真的存在周期，以及周期对公司盈利是否重要。

有些行业是弱周期，有些行业是多周期叠加，有些所谓周期其实是结构性变化。

## 2. 周期不一定规律

必须遵守：

- 不要假设所有行业都有周期。
- 不要假设周期一定往返。
- 不要假设周期频率固定。
- 不要只根据历史均值判断当前周期。
- 不要把短期价格波动误判为行业周期。
- 不要把结构性衰退误判为周期底部。

周期可能是规律波动、不规则波动、结构性上行、结构性下行、一次性冲击、政策变化、技术替代或供给出清。

## 3. 行业周期类型

| 类型 | 示例 |
|---|---|
| 非周期/弱周期 | 必选消费、公用事业、部分医疗服务、稳定软件 |
| 传统周期 | 煤炭、有色、钢铁、化工、航运、造纸、工程机械 |
| 库存周期 | 半导体、消费电子、汽车、家电、工业品 |
| 产能周期 | 光伏、新能源中上游、化工、钢铁、航运、面板 |
| 利率/信用周期 | 银行、地产、保险、券商、建筑建材 |
| 政策周期 | 教育、医药、军工、新能源、环保、地产链 |
| 技术周期 | AI、半导体、云计算、机器人、创新药、智能汽车 |
| 消费周期 | 可选消费、奢侈品、餐饮旅游、广告 |
| 监管周期 | 互联网平台、博彩、医药、金融科技 |
| 多周期叠加 | 半导体、智能汽车、新能源、互联网平台 |

## 4. 多周期叠加

一个行业可以同时受多个周期影响。  
例如半导体可能同时受：

- 库存周期
- 资本开支周期
- 技术周期
- 地缘政策周期
- AI 或终端需求周期

分析时要分别拆开，而不是只说“半导体处于周期底部”。

## 5. 周期阶段

可以使用以下阶段，但不能机械套用：

- 结构性成长
- 景气上行
- 景气高位
- 边际转弱
- 下行
- 筑底
- 早期复苏
- 结构性衰退
- 无法判断

每个阶段必须有证据支持。

## 6. 关键指标

### 供需和库存

需求增速、供给增速、库存、订单、开工率、产能利用率、进出口。

### 价格和利润

产品价格、原材料价格、价差、吨盈利、单位盈利、毛利率。

### 产能和资本开支

资本开支、在建工程、新增产能、投产节奏、淘汰产能、行业亏损比例。

### 政策和信用

政策文件、补贴、审批、监管态度、利率、信贷、社融、融资成本、违约率。

### 技术和消费

技术路线、渗透率、替代速度、客户采用、消费信心、客流、客单价、广告预算。

## 7. 公司周期敏感度

不是所有公司都同等受周期影响。

敏感度分为：

- 高度敏感
- 中度敏感
- 低度敏感
- 反周期

判断因素：

- 产业链位置
- 定价权
- 成本转嫁能力
- 客户结构
- 合约锁价
- 长期订单
- 库存管理
- 业务多元化
- 资产负债表强弱

## 8. 周期对估值的影响

周期顶部：低 P/E 可能是陷阱。  
周期底部：高 P/E 不一定贵。  
景气上行：盈利和估值倍数可能双升。  
景气下行：盈利和估值倍数可能双杀。  
结构性衰退：不能期待简单均值回归。

估值建议：

- 周期公司要看正常化盈利。
- DCF 应使用中周期利润率和现金流。
- 相对估值应避免直接使用峰值或谷底利润。
- 周期极端位置要提高风险提示。

## 9. 跨市场重点

### A 股

关注政策周期、产业链景气、库存、价格、产能、国产替代、补贴、环保能耗约束和主题资金。

### 港股

关注中国宏观、地产周期、消费周期、金融周期、资源价格、南向资金和外资风险偏好。

### 美股

关注利率周期、库存周期、科技资本开支、消费数据、PMI、就业、美元、全球供应链和全球龙头指引。

### 中概股

关注中国需求、海外上市估值、汇率、监管周期和地缘风险。

## 10. 数据不足原则

- 缺少行业价格和库存，不能高置信判断商品/周期行业位置。
- 缺少产能数据，不能高置信判断供给周期。
- 缺少政策信息，不能高置信判断政策周期。
- 缺少历史行业数据，不能判断周期是否规律。
- 缺少公司分业务收入，不能可靠判断公司周期敏感度。
