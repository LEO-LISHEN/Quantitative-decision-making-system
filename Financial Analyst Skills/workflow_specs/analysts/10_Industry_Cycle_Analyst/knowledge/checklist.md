# 检查清单

## 行业识别

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认主要行业和业务板块？
- [ ] 是否确认信息截止时间？
- [ ] 多业务公司是否分业务判断周期？

## 是否存在周期

- [ ] 是否先判断行业是否真的存在周期？
- [ ] 是否避免默认所有行业都有周期？
- [ ] 是否说明周期对公司盈利是否重要？
- [ ] 如果是弱周期或非周期，是否明确说明？

## 周期类型

- [ ] 是否判断传统周期？
- [ ] 是否判断库存周期？
- [ ] 是否判断产能周期？
- [ ] 是否判断利率/信用周期？
- [ ] 是否判断政策周期？
- [ ] 是否判断技术周期？
- [ ] 是否判断消费周期？
- [ ] 是否判断监管周期？
- [ ] 是否判断多周期叠加？

## 周期是否规律

- [ ] 是否判断周期是否规律？
- [ ] 是否避免假设固定频率？
- [ ] 是否识别结构性上行或结构性下行？
- [ ] 是否识别一次性冲击、政策变化或技术替代？

## 关键数据

- [ ] 是否分析需求？
- [ ] 是否分析供给？
- [ ] 是否分析库存？
- [ ] 是否分析价格？
- [ ] 是否分析产能和产能利用率？
- [ ] 是否分析订单？
- [ ] 是否分析资本开支？
- [ ] 是否分析政策、利率或技术变量？

## 景气阶段

- [ ] 是否判断结构性成长、景气上行、景气高位、边际转弱、下行、筑底、早期复苏或结构性衰退？
- [ ] 是否提供证据？
- [ ] 是否列出领先指标？
- [ ] 是否列出滞后指标？

## 公司敏感度

- [ ] 是否判断公司对行业周期的敏感度？
- [ ] 是否分析产业链位置？
- [ ] 是否分析定价权和成本转嫁？
- [ ] 是否分析合同锁价或长期订单？
- [ ] 是否分析资产负债表韧性？

## 估值影响

- [ ] 是否判断当前利润是否为周期高点或低点？
- [ ] 是否说明是否需要正常化盈利？
- [ ] 是否给 DCF 提供中周期假设？
- [ ] 是否提醒相对估值避免峰值/谷底倍数？
- [ ] 是否列出周期风险和反证信号？

## 跨市场

- [ ] A 股是否关注政策、产业链、库存、价格、产能、补贴和主题资金？
- [ ] 港股是否关注中国宏观、地产、消费、金融、资源和南向资金？
- [ ] 美股是否关注利率、库存、科技资本开支、PMI、就业、美元和全球供应链？
- [ ] 中概股是否关注中国需求、汇率、监管和地缘风险？

## 输出

- [ ] 是否避免直接给最终买卖建议？
- [ ] 是否说明数据缺口和置信度？
- [ ] 是否给出后续跟踪指标？
