# 检查清单

## 公司和市场

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认估值币种？
- [ ] 是否确认信息截止时间？
- [ ] 是否识别市场估值风格：A 股、港股、美股或中概股？

## 市场估值锚

- [ ] 是否判断市场主要使用什么估值锚？
- [ ] 是否说明为什么使用该估值锚？
- [ ] 是否列出辅助估值锚？

## 可比公司

- [ ] 是否列出可比公司？
- [ ] 是否说明纳入原因？
- [ ] 是否说明排除原因或限制？
- [ ] 是否给出可比性评级？
- [ ] 是否检查业务模式相似性？
- [ ] 是否检查增长阶段和利润率？
- [ ] 是否检查区域市场和监管环境？
- [ ] 是否检查市值和流动性？

## 倍数选择

- [ ] 是否选择适合行业的估值倍数？
- [ ] 是否避免用不适合的倍数？
- [ ] 是否考虑 P/E、Forward P/E、P/B、EV/EBITDA、P/S、P/FCF、股息率或行业倍数？
- [ ] 多业务公司是否考虑 SOTP？

## 口径校验

- [ ] 是否区分静态、TTM 和 Forward？
- [ ] 是否区分 GAAP、non-GAAP、归母、扣非和 adjusted？
- [ ] 是否正确计算 EV？
- [ ] 是否检查净债务、现金和少数股东权益？
- [ ] 是否使用最新股本？
- [ ] 是否剔除一次性项目？
- [ ] 是否统一币种？
- [ ] 是否考虑会计准则差异？

## 溢价折价

- [ ] 是否判断应溢价、平价或折价？
- [ ] 是否说明增长、ROIC、现金流、护城河和财务风险影响？
- [ ] 是否说明治理、流动性、客户集中和周期风险影响？

## 历史估值

- [ ] 是否分析 3 年、5 年或 10 年历史估值区间？
- [ ] 是否说明当前历史分位？
- [ ] 是否判断历史区间是否仍可比？

## 跨市场

- [ ] A 股是否考虑主题、政策、流动性、小盘、红利和景气度？
- [ ] 港股是否考虑流动性折价、治理折价、控股股东、南向资金和 NAV 折价？
- [ ] 美股是否考虑 Forward multiples、FCF yield、Rule of 40、SBC、回购和全球龙头溢价？
- [ ] 中概股是否考虑 ADR/H 股折价、VIE、监管、审计和地缘风险？

## 输出

- [ ] 是否输出合理倍数区间？
- [ ] 是否输出隐含股权价值和每股价值前提？
- [ ] 是否与 DCF 或其他估值方法交叉验证？
- [ ] 如果没有可靠可比公司，是否明确说明限制？
- [ ] 是否避免直接给最终买卖建议？
