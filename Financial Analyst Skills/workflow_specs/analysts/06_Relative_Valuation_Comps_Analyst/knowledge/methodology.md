# 方法论

## 1. 相对估值的本质

相对估值不是简单找同行平均倍数。  
它回答的是：

- 市场愿意为类似资产支付多少价格？
- 这家公司相对同行应当溢价还是折价？
- 当前市场更可能使用什么估值锚？

## 2. 可比公司选择

好的可比公司必须在多个维度相似：

- 业务模式
- 收入来源
- 利润率
- 增长阶段
- 区域市场
- 客户结构
- 资本结构
- 周期属性
- 监管环境
- 市值和流动性
- 会计准则

如果可比公司只是行业标签相同，但商业模式不同，就不能直接套倍数。

## 3. 倍数选择

| 公司类型 | 常用倍数 |
|---|---|
| 稳定盈利公司 | P/E、Forward P/E、P/FCF |
| 亏损成长公司 | EV/Sales、P/S、远期收入倍数 |
| 周期股 | EV/EBITDA、P/B、正常化 P/E |
| 银行 | P/B + ROE |
| 保险 | P/EV、P/B、新业务价值 |
| 地产 | NAV 折价、P/B |
| 资源股 | EV/EBITDA、NAV、储量价值 |
| SaaS | EV/ARR、EV/Sales、Rule of 40 |
| 平台公司 | P/E、EV/EBITDA、SOTP |
| 高股息资产 | 股息率、P/E、P/B |

## 4. 倍数口径校验

倍数必须校验口径：

- 静态、TTM、Forward。
- GAAP、non-GAAP、归母、扣非、adjusted earnings。
- EV 是否正确计算。
- 是否包含少数股东权益。
- 股本是否最新。
- 是否剔除一次性项目。
- 币种是否统一。
- 会计准则是否可比。
- 财务红旗是否影响利润或现金流口径。

口径不一致的倍数不能直接比较。

## 5. 溢价和折价

公司应享受溢价的常见原因：

- 增长更快。
- ROIC 更高。
- 现金流更好。
- 护城河更强。
- 财务风险更低。
- 管理层资本配置更好。
- 流动性更好。
- 市场地位更强。

公司应折价的常见原因：

- 增长较慢。
- 利润率较低。
- 治理较差。
- 财务红旗。
- 客户集中。
- 周期性强。
- 流动性差。
- 政策或监管风险高。

## 6. 历史估值区间

历史估值有用，但不是永远有效。  
使用历史估值时要检查：

- 公司商业质量是否变化。
- 行业景气度是否可比。
- 利率和流动性环境是否可比。
- 投资者结构是否变化。
- 公司是否发生重大业务转型。

历史低位不一定便宜，历史高位也不一定贵。

## 7. SOTP 分部估值

多业务公司应考虑分部估值：

- 核心业务用经营倍数。
- 投资资产按市值或折价估值。
- 亏损业务用概率或折价处理。
- 金融资产、现金、净债务单独调整。
- 控股平台可考虑控股折价。

适用对象包括互联网平台、综合集团、控股公司、资源集团和持有大量投资资产的公司。

## 8. 跨市场估值风格

### A 股

A 股估值常受政策、主题、流动性、景气度和交易拥挤度影响。  
可能出现主题溢价、小盘溢价、国企改革溢价和红利风格切换。

### 港股

港股常见流动性折价、治理折价、控股股东折价和 NAV 折价。  
高股息、回购和南向资金可能推动估值修复。

### 美股

美股更常使用 Forward multiples、FCF yield、EV/Sales、Rule of 40 和行业运营倍数。  
高 ROIC、全球龙头、回购和长期增长可获得溢价。

### 中概股

中概股需要考虑 ADR/H 股折价、VIE、监管、审计、地缘风险、汇率和流动性差异。

## 9. 无可靠可比公司的处理

如果没有足够好的可比公司，应明确说明：

- 相对估值只能作为辅助参考。
- 不应机械套用行业平均值。
- 应考虑 DCF、SOTP、NAV、DDM、剩余收益或情景估值。

这不是失败，而是正确的估值纪律。
