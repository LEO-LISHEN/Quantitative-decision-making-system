# 检查清单

## 基础信息

- [ ] 公司名称、股票代码、市场和交易所已确认。
- [ ] 分析日期和信息截止时间已确认。
- [ ] 被检验的多头逻辑已明确。
- [ ] 估值假设、成长假设和催化剂假设已明确。

## 来源检查

- [ ] 已检查年报、半年报、季报或 10-K/10-Q/20-F。
- [ ] 已检查审计意见和内控披露。
- [ ] 已检查财务报表附注。
- [ ] 已检查监管问询、处罚和交易所函件。
- [ ] 已检查诉讼、仲裁或法律文件。
- [ ] 已检查做空报告和公司回应，如存在。
- [ ] 已检查新闻、媒体调查和第三方报告。
- [ ] 已检查价格、成交量、相对强弱或做空数据，如可得。
- [ ] 已检查舆情、投诉、员工评价或用户评价中的风险线索。

## 证据分级

- [ ] 已区分事实、指控、推断、传闻和情绪表达。
- [ ] 已标注证据等级：已证实、高可信、待验证、低可信、已证伪。
- [ ] 已标注来源、日期和原始材料。
- [ ] 未把未经验证的传闻写成事实。

## 风险地图

- [ ] 已检查商业模式风险。
- [ ] 已检查财务质量风险。
- [ ] 已检查会计和审计风险。
- [ ] 已检查监管和法律风险。
- [ ] 已检查治理风险。
- [ ] 已检查竞争和技术风险。
- [ ] 已检查资本结构和流动性风险。
- [ ] 已检查估值风险。
- [ ] 已检查市场技术风险。
- [ ] 已检查情绪舆情风险。
- [ ] 已检查行业周期和宏观风险。

## 反证矩阵

- [ ] 每个关键假设都有支持证据。
- [ ] 每个关键假设都有反证或削弱证据。
- [ ] 每个关键假设都有早期预警指标。
- [ ] 每个关键假设都有逻辑失效触发条件。
- [ ] 已判断当前状态：仍成立、被削弱、高风险、已失效或无法判断。

## 下行情景

- [ ] 已建立悲观情景。
- [ ] 已建立压力情景。
- [ ] 已说明收入影响。
- [ ] 已说明利润率影响。
- [ ] 已说明现金流影响。
- [ ] 已说明估值倍数或折现率影响。
- [ ] 已说明融资、稀释或流动性影响。

## 做空视角

- [ ] 已提出最强反方论点。
- [ ] 已列出支持反方的证据。
- [ ] 已列出反方论点的弱点。
- [ ] 已记录公司回应或澄清。
- [ ] 已避免把做空报告指控直接当事实。

## 交付边界

- [ ] 已输出总体风险等级。
- [ ] 已输出最重要风险。
- [ ] 已输出可能已定价和未充分定价的风险。
- [ ] 已列出最终建议前必须核验的事项。
- [ ] 已列出需要其他分析师复核的问题。
- [ ] 没有直接给最终买入、卖出、持有或做空建议。
