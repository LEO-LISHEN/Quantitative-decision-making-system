# 检查清单

## 数据完整性

- [ ] 是否获取最近 3 年年报？
- [ ] 是否获取最近 4 个季度财报？
- [ ] 是否尽量获取最近 5 年年报和 8 个季度数据？
- [ ] 是否有利润表？
- [ ] 是否有资产负债表？
- [ ] 是否有现金流量表？
- [ ] 是否有附注？
- [ ] 是否有审计意见？
- [ ] 是否有分部信息？
- [ ] 是否有管理层讨论与分析？

## 口径校验

- [ ] 是否确认币种？
- [ ] 是否确认单位？
- [ ] 是否确认报告期？
- [ ] 是否确认会计准则？
- [ ] 是否确认合并范围？
- [ ] 是否区分法定口径和调整后口径？
- [ ] 是否区分 TTM、年度、季度和预测数据？
- [ ] 是否检查非经常性项目？
- [ ] 是否检查重述和会计政策变更？
- [ ] 是否检查股本变化和汇率影响？

## 历史趋势

- [ ] 是否分析收入趋势？
- [ ] 是否分析毛利率趋势？
- [ ] 是否分析经营利润率趋势？
- [ ] 是否分析净利率趋势？
- [ ] 是否分析 ROE/ROIC 趋势？
- [ ] 是否分析经营现金流趋势？
- [ ] 是否分析自由现金流趋势？
- [ ] 是否分析资本开支趋势？
- [ ] 是否分析费用率趋势？
- [ ] 是否分析应收和存货周转？

## 盈利质量

- [ ] 净利润是否有经营现金流支持？
- [ ] 收入增长是否与应收账款增长匹配？
- [ ] 毛利率变化是否有业务解释？
- [ ] 利润是否依赖非经常性损益？
- [ ] 调整后利润是否能与法定利润勾稽？
- [ ] 自由现金流是否支持分红和回购？

## 资产负债表

- [ ] 应收账款是否异常？
- [ ] 存货是否异常？
- [ ] 合同资产是否异常？
- [ ] 预付款是否异常？
- [ ] 商誉和无形资产是否过高？
- [ ] 在建工程是否长期不转固？
- [ ] 短债和流动性是否安全？
- [ ] 是否存在或有负债、担保或关联方占款？

## 发展计划财务验证

- [ ] 扩产计划是否对应资本开支和在建工程？
- [ ] 研发计划是否对应研发费用率和资本化比例？
- [ ] 海外扩张是否影响汇率、应收和毛利率？
- [ ] 并购是否带来商誉和整合风险？
- [ ] 渠道扩张是否推高销售费用率和存货？
- [ ] 分红回购是否由自由现金流支持？

## 跨市场

- [ ] A 股是否检查扣非净利润、非经常性损益、问询函、应收、存货和商誉？
- [ ] 港股是否检查公允价值、分红能力、净债务、少数股东权益和持续经营？
- [ ] 美股是否检查 GAAP/non-GAAP、SBC、递延收入、FCF、回购和稀释？

## 输出

- [ ] 是否先说明数据完整性？
- [ ] 是否列出财务红旗并分级？
- [ ] 是否给出正常化利润或现金流判断？
- [ ] 是否说明对估值的影响？
- [ ] 是否避免直接给最终买卖建议？
