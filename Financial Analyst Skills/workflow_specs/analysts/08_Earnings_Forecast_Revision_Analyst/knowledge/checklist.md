# 检查清单

## 数据完整性

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认信息截止时间？
- [ ] 是否有历史收入、利润率、费用率、EPS 和现金流？
- [ ] 是否有一致预期？
- [ ] 是否有管理层指引？
- [ ] 是否有业绩预告、业绩快报或盈利警告？
- [ ] 是否有订单、价格、成本、库存或产能数据？
- [ ] 是否有同行财报和指引？

## 盈利基准

- [ ] 是否建立收入基准？
- [ ] 是否建立毛利率基准？
- [ ] 是否建立经营利润率基准？
- [ ] 是否建立净利润或 EPS 基准？
- [ ] 是否建立自由现金流基准？
- [ ] 是否说明预测口径？

## 一致预期和指引

- [ ] 是否提取收入 consensus？
- [ ] 是否提取 EPS consensus？
- [ ] 是否提取管理层收入指引？
- [ ] 是否提取管理层利润率指引？
- [ ] 是否提取分析师上修或下修记录？
- [ ] 不同来源冲突时是否标注？

## 收入驱动

- [ ] 是否分析销量？
- [ ] 是否分析价格？
- [ ] 是否分析订单和 backlog？
- [ ] 是否分析产能和利用率？
- [ ] 是否分析用户数、ARPU、订阅、GMV 或广告指标？
- [ ] 是否分析汇率影响？

## 利润率和 EPS 驱动

- [ ] 是否分析产品结构？
- [ ] 是否分析成本和原材料？
- [ ] 是否分析费用率？
- [ ] 是否分析折旧、研发和 SBC？
- [ ] 是否分析税率、利息、汇兑和股本变化？
- [ ] 是否分析回购或稀释？

## 现金流质量

- [ ] 盈利上修是否有经营现金流支持？
- [ ] 营运资本是否恶化？
- [ ] 资本开支是否压低自由现金流？
- [ ] 回款和库存是否支持收入增长？

## 修正判断

- [ ] 是否分别判断收入、利润率、净利润/EPS 和自由现金流？
- [ ] 是否判断上修、小幅上修、稳定、小幅下修、下修或高度不确定？
- [ ] 是否判断当前季度、下一季度、未来 4 个季度和未来 2-3 年？
- [ ] 是否判断修正质量？
- [ ] 是否列出领先指标？

## 跨市场

- [ ] A 股是否关注业绩预告、扣非净利润、订单、价格、机构调研和高频数据？
- [ ] 港股是否关注盈利警告、业绩公告、分红、减值、公允价值和净债务？
- [ ] 美股是否关注 consensus、guidance、analyst revision、non-GAAP、SBC、ARR/RPO 和 deferred revenue？
- [ ] 中概股是否关注汇率、non-GAAP 和 ADR/港股差异？

## 输出

- [ ] 是否说明缺失数据和置信度影响？
- [ ] 是否说明对 DCF 和相对估值的影响？
- [ ] 是否说明应传递给预期差分析师的信息？
- [ ] 是否避免直接给最终买卖建议？
