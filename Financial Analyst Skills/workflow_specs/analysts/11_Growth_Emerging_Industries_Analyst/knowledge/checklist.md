# 检查清单

## 基础信息

- [ ] 公司名称、股票代码、交易市场、行业、分析日期已确认。
- [ ] 已确认报告货币、会计口径和信息截止日期。
- [ ] 已区分 A 股、港股、美股或中概股的披露差异。

## 来源检查

- [ ] 已检查最近年报。
- [ ] 已检查最近半年报/中报或季度报告。
- [ ] 已检查公告、投资者关系材料、业绩会或电话会纪要。
- [ ] 已检查行业报告、研报或第三方数据。
- [ ] 已检查新闻和行业媒体。
- [ ] 如使用社交媒体、股吧、雪球、X/Reddit 或论坛，已标注其为线索而非高置信证据。
- [ ] 已记录来源发布时间，避免把旧信息误判为新成长逻辑。
- [ ] 已列出缺失关键资料。

## 成长叙事

- [ ] 已用一句话说明核心成长故事。
- [ ] 已说明叙事来源于公司披露、行业趋势、研报、新闻、社交媒体还是市场主题。
- [ ] 已判断成长故事是证据支持、部分支持、主要叙事还是无法判断。
- [ ] 已区分行业增长、公司份额提升和价格变化。

## 市场空间

- [ ] 已区分 TAM、SAM、SOM。
- [ ] 已检查市场规模口径。
- [ ] 已检查渗透率和采用阶段。
- [ ] 已分析公司可触达市场和可获得份额。
- [ ] 已避免把巨大 TAM 直接当作估值依据。

## 公司验证

- [ ] 已检查产品或技术是否商业化。
- [ ] 已检查客户、订单、用户、留存、复购、续费或产能利用率。
- [ ] 已检查竞争对手和替代技术。
- [ ] 已核对管理层规划与实际投入、收入和现金流是否一致。
- [ ] 已判断护城河来自技术、成本、渠道、品牌、数据、生态、牌照还是客户关系。

## 单位经济模型

- [ ] 已选择行业适用指标。
- [ ] 已检查毛利率路径。
- [ ] 已检查获客或销售效率。
- [ ] 已检查资本开支和研发投入。
- [ ] 已检查现金消耗和融资需求。
- [ ] 已判断增长是否具备规模效应。

## 情景与估值

- [ ] 已建立悲观、基准、乐观情景。
- [ ] 每个情景都有市场份额、收入、利润率、现金流或估值假设。
- [ ] 已为情景标注概率或概率范围。
- [ ] 已判断适合远期倍数、SOTP、DCF、概率加权、实物期权或暂不适合高置信估值。
- [ ] 已避免只用乐观情景估值。

## 风险与证伪

- [ ] 已列出执行风险。
- [ ] 已列出竞争风险。
- [ ] 已列出技术风险。
- [ ] 已列出监管风险。
- [ ] 已列出融资和稀释风险。
- [ ] 已列出估值压缩风险。
- [ ] 已列出未来能证伪成长故事的信号。

## 交付给总控分析师

- [ ] 已给出成长质量判断。
- [ ] 已给出成长兑现概率。
- [ ] 已给出对估值溢价或折价的影响。
- [ ] 已给出最终建议前必须补充验证的事项。
- [ ] 没有直接给最终买入、卖出或持有建议。
