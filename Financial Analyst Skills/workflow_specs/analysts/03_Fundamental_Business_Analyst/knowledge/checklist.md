# 检查清单

## 公司和市场

- [ ] 是否确认公司名称和股票代码？
- [ ] 是否确认上市市场和交易所？
- [ ] 是否确认信息截止时间？
- [ ] 是否识别 A 股、港股、美股、中概股或多地上市语境？

## 商业模式

- [ ] 是否说明公司卖什么？
- [ ] 是否说明客户是谁？
- [ ] 是否说明客户为什么买？
- [ ] 是否说明收入模式？
- [ ] 是否说明利润模式？
- [ ] 是否说明现金流特征？
- [ ] 是否按业务板块拆分，而不是只看整体？

## 增长质量

- [ ] 是否区分结构性增长、周期性增长和一次性增长？
- [ ] 是否识别价格、销量、份额、并购、补贴或会计口径影响？
- [ ] 是否判断增长能否转化为利润和现金流？

## 竞争格局

- [ ] 是否识别主要竞争对手？
- [ ] 是否判断竞争强度？
- [ ] 是否分析客户议价能力？
- [ ] 是否分析供应商议价能力？
- [ ] 是否分析替代品和新进入者风险？
- [ ] 是否识别价格战或产能过剩风险？

## 护城河

- [ ] 是否识别护城河来源？
- [ ] 是否判断护城河强度？
- [ ] 是否判断护城河趋势：变宽、稳定、变窄或不确定？
- [ ] 是否用利润率、ROIC、份额或客户行为验证护城河？

## ROIC 与再投资

- [ ] 是否判断 ROIC 是否可能高于资本成本？
- [ ] 是否判断公司是否有再投资空间？
- [ ] 是否判断增长是否需要大量资本开支？
- [ ] 是否判断规模扩大后利润率是否改善？

## 管理层和资本配置

- [ ] 是否评估回购、分红、并购、扩产、研发或融资？
- [ ] 是否识别关联交易或损害小股东风险？
- [ ] 是否判断管理层历史承诺是否兑现？

## 跨市场

- [ ] A 股公司是否检查政策、产业链、招投标、国资、补贴、机构调研？
- [ ] 港股公司是否检查治理、分红、控股股东、流动性折价、南向资金？
- [ ] 美股公司是否检查全球竞争力、平台生态、SBC、回购、长期叙事？
- [ ] 中概股或多地上市公司是否区分主要经营地、披露地和定价市场？

## 输出

- [ ] 是否给出商业质量评级？
- [ ] 是否给出估值含义？
- [ ] 是否列出需要下游分析师验证的问题？
- [ ] 是否避免直接给最终买卖建议？
