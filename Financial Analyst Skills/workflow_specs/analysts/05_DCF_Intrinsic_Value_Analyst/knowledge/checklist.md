# 检查清单

## DCF 适用性

- [ ] 是否先判断 DCF 适用性？
- [ ] 是否输出适合、部分适合、暂不适合或不适合？
- [ ] 如果不适合，是否停止硬建模？
- [ ] 是否说明不适合的原因？
- [ ] 是否推荐替代估值方法？

## 数据完整性

- [ ] 是否有历史收入和利润率？
- [ ] 是否有经营现金流？
- [ ] 是否有资本开支？
- [ ] 是否有营运资本变化？
- [ ] 是否有折旧摊销？
- [ ] 是否有净债务或净现金？
- [ ] 是否有股本数量？
- [ ] 是否有税率？
- [ ] 是否有管理层指引或一致预期？
- [ ] 是否有折现率相关数据？

## 现金流假设来源

- [ ] 是否使用财务分析师输出？
- [ ] 是否使用基本面分析师输出？
- [ ] 是否使用行业周期信息？
- [ ] 是否使用盈利预测或管理层指引？
- [ ] 是否说明主动搜集的数据来源？
- [ ] 是否避免无来源假设？

## 模型选择

- [ ] 是否说明选择 FCFF、FCFE、DDM、剩余收益、情景 DCF 或不建模的原因？
- [ ] 金融公司是否避免机械使用普通 FCFF？
- [ ] 高股息公司是否考虑 DDM？
- [ ] 银行保险是否考虑剩余收益或 P/B/ROE？
- [ ] 周期公司是否考虑正常化盈利或情景法？

## 三情景

- [ ] 是否有悲观情景？
- [ ] 是否有基准情景？
- [ ] 是否有乐观情景？
- [ ] 每个情景是否说明收入增长、利润率、资本开支、营运资本和终值？
- [ ] 每个情景是否说明触发条件？

## 敏感性

- [ ] 是否分析 WACC？
- [ ] 是否分析永续增长率？
- [ ] 是否分析退出倍数？
- [ ] 是否分析长期利润率？
- [ ] 是否分析收入增速？
- [ ] 是否分析资本开支率？
- [ ] 是否分析营运资本占用？

## 终值风险

- [ ] 是否披露终值占企业价值比例？
- [ ] 如果终值占比过高，是否降低置信度？
- [ ] 永续增长率是否合理？
- [ ] 退出倍数是否与同行和公司质量匹配？

## 跨市场

- [ ] A 股是否考虑人民币利率、政策风险、监管和流动性？
- [ ] 港股是否考虑港元/美元利率、流动性折价、治理折价和分红？
- [ ] 美股是否考虑美元利率、ERP、Beta、SBC、回购和全球业务？
- [ ] 中概股是否考虑 VIE、审计监管、汇率和地缘风险？

## 输出

- [ ] 是否避免单点目标价？
- [ ] 是否输出企业价值、股权价值和每股价值的前提？
- [ ] 数据不足时是否标注无法可靠计算？
- [ ] 是否说明最敏感和最可能出错的假设？
- [ ] 是否避免直接给最终买卖建议？
