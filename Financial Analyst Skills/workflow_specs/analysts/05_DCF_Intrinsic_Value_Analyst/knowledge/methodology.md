# 方法论

## 1. DCF 不是万能方法

DCF 的本质是把未来现金流折现到现在。  
如果未来现金流不可预测，DCF 就会变成假设游戏。

因此，DCF 分析师必须先判断适用性：

- 适合
- 部分适合
- 暂不适合
- 不适合

不适合时，明确拒绝硬建模比输出虚假目标价更专业。

## 2. 适用性判断

适合 DCF 的公司通常具备：

- 商业模式清晰。
- 财务数据足够。
- 现金流相对稳定。
- 收入和利润率可预测。
- 资本开支和营运资本可估计。
- 债务和股本结构可识别。
- 长期竞争力有证据支持。

不适合 DCF 的情况：

- 持续亏损且商业模式未验证。
- 现金流高度波动。
- 周期极端位置。
- 数据不足。
- 并购重组导致业务边界变化。
- 创新药早期管线、矿产勘探、困境重组和壳公司。
- 普通 FCFF 不适合的银行、保险和部分金融机构。

## 3. 替代估值方法

| 公司类型 | 更适合的方法 |
|---|---|
| 周期股 | 正常化盈利、EV/EBITDA、P/B |
| 银行 | P/B、ROE、剩余收益 |
| 保险 | P/EV、新业务价值、剩余收益 |
| 地产 | NAV、RNAV、SOTP |
| 资源股 | NAV、储量价值、周期正常化估值 |
| 早期创新药 | 概率加权管线估值 |
| SaaS 高成长 | 远期收入倍数、Rule of 40、情景 DCF |
| 亏损成长股 | 远期市值折现、概率情景 |
| 控股平台 | SOTP |
| 高股息资产 | DDM、股息收益率 |
| 困境重组 | 清算价值、情景估值 |

## 4. 现金流预测来源

DCF 假设必须来自证据：

- 财务分析师提供历史现金流、资本开支、营运资本、债务和股本。
- 基本面分析师提供商业质量、护城河、ROIC 和再投资空间。
- 行业周期分析师提供行业增速、价格周期和供需变化。
- 盈利预测分析师提供未来收入、利润率、订单、成本和管理层指引。
- 信息源分析师提供公告、研报、新闻和资本开支计划。
- 外部数据提供利率、ERP、Beta、债务成本和同行资本结构。

## 5. 模型选择

- FCFF：适合多数非金融企业。
- FCFE：适合股权现金流稳定、资本结构较清晰的公司。
- DDM：适合高分红和成熟现金回报资产。
- 剩余收益：适合银行、保险和账面价值有意义的公司。
- 情景 DCF：适合不确定性高但仍能构建情景的公司。
- 不建模：适合性不足、数据缺失或现金流无法预测时。

## 6. 三情景估值

DCF 不应只给一个基准值。  
必须建立悲观、基准、乐观三种情景，并说明触发条件。

常见情景变量：

- 收入增长
- 利润率
- 税率
- 资本开支
- 营运资本
- 自由现金流
- 终值增长率或退出倍数
- 折现率

## 7. 敏感性分析

必须测试：

- WACC
- 永续增长率
- 退出倍数
- 长期利润率
- 收入增速
- 资本开支率
- 营运资本占用

敏感性分析的目的不是制造表格，而是找出“哪个假设一错，估值就会大幅变化”。

## 8. 终值风险

DCF 很容易被终值主导。  
必须披露终值占企业价值比例。

风险提示：

- 终值占比越高，估值越依赖长期假设。
- 永续增长率不能明显高于长期经济增长，除非有充分证据。
- 退出倍数不能脱离同行估值和公司质量。
- 如果终值支撑了大部分估值，应降低结论置信度。

## 9. 跨市场折现率

### A 股

考虑人民币利率、政策风险、行业监管、国企属性、流动性和市场波动。

### 港股

考虑港元/美元利率、流动性折价、治理折价、控股股东、南向资金、分红政策和资产折价。

### 美股

考虑美元利率、ERP、Beta、全球业务风险、SBC 稀释、回购和长期增长溢价。

### 中概股

额外考虑 VIE、审计监管、地缘政治、ADR/港股双重上市和汇率风险。

## 10. 输出原则

- DCF 结果是区间，不是确定价格。
- 数据不足时，优先输出假设地图。
- 不适合 DCF 时，明确拒绝建模。
- 每股价值只有在净债务、少数股东权益、股本数量足够可靠时才输出。
- DCF 不能替代相对估值、预期差和催化剂分析。
